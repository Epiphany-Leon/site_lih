# 晋江

自从爱上女频之后，从耽美到言情，再到百合最后无 CP，虽说频道分类不会影响文本身质量，但是会让我有所偏好

## 纯爱

- [ ] 宋玉章（朋友，这一本书写的太好了，写出了真正的民国
- [ ] 我是龙傲天他惨死的爹【穿书】（水仙；很多破境的感悟写的很牛逼；大部分剧情是男频爽文土狗剧情，我是土狗我最爱
- [ ] 千秋（我的顺序：动漫——广播剧——小说，正道主角真的好难刻画的
- [ ] 成为暴君之后（重构中世纪黑暗时期的历史向文，很喜欢这位作者
- [ ] 美人挑灯看剑（重构中国上古时期历史向文，很多古文夹杂其中，好处是我能看懂且能带入，坏处是行文有些晦涩割裂
- [ ] priest（我好爱皮皮，理想主义永不死
	- [ ] 杀破狼
	- [ ] 太岁（这个好像是无 CP 向的
	- [ ] 大哥
	- [ ] 六爻
- [ ] 花滑不是爽文所以不能一蹴而就（虽然不好 ky，但我就是带入国服羽生结弦看的，进入奥运周期主角还带有亚古丁、兰比尔、普鲁申科的影子😭 这才是弥补现实遗憾的【天降紫微星】，什么时候种花家能有这样的培养体系。。感谢天天&葱桶现役🙏
- [ ] 死亡万花筒
- [ ] 营业悖论【娱乐圈】（虽然犯了娱乐圈爱豆的大忌👊 奈何广播剧太好了
- [ ] 独立电影人（对电影的理解很强，让我接触认识到了很多新知识。如果没有后期的做来做去，我可能会更喜欢它
- [ ] 将进酒（18 岁的唐酒卿，太牛逼了
- [ ] 定海浮生录
- [ ] 天官赐福（墨香三部曲都挺好的，我的顺序是：渣反，魔道，天官。天官是最像正剧的，渣反是最有灵气的
- [ ] 灰塔笔记（这是我在晋江读的第一本书吧，哪怕剧情记不住，印象也是非常深刻

## 言情

- [ ] 御兽王座（女频中少见的御兽文，且走的是暗黑 if 线；阴差阳错，命运使然，这该死的世道与宿命感
- [ ] 香江大厨【八零】（很容易看饿的一本书；营销学实例
- [ ] 穿成科举文里的嫡长孙（我超爱这本书♥️ 这样的穿越科举文，难得啊！！
- [ ] 社稷山河剑（设定和剧情走向都很对我胃口，可惜作者坑了
- [ ] 身为女王如何拒绝爱意（雄竞，爱看；重构大英工业革命的历史向文，结局女王嫁给自己的国家，我太喜欢这个结局了
- [ ] 女王的野兽（雄竞，爱看
- [ ] 天才基本法（虽然我不喜欢数学，但我喜欢书中描写的“一以贯之的努力”
- [ ] 高危职业二师姐
- [ ] 我靠亲爸在虐文破案【刑侦】
- [ ] 三婚（一婚更比一婚高。写出了年上的顾虑和年下的无畏
- [ ] 尾鱼（全肯定，她的书真的都好看！设定很完备，脑洞也很大

## 百合

- [ ] 穿为帝国皇女的渣 A 前妻（剧情就那样吧，吸引我的点在两人感情的恨海情天与双重酸涩；中后期有点酸涩过头了，会有些不适
- [ ] 今日离港（因为广播剧去看的小说
